Limpiador de Texto

**Descripción:**

Elimina palabras de un archivo de texto según patrones definidos por el usuario.

**Uso:**

1. Ejecutar el script.
2. Ingresar la ruta del archivo a limpiar.
3. Ingresar el nombre del archivo de salida.
4. Separar los patrones a eliminar por comas.

**Ejemplo:**

python limpia_texto.py
Ruta del archivo: mi_archivo
Nombre del archivo de salida: resultado
Patrones: man,apple,banana

**Funcionalidades:**

* Genera expresiones regulares automáticamente.
* Maneja múltiples patrones.
* Preserva la estructura del texto original.

**Nota:**

* Para personalizar la búsqueda, modifica la expresión regular.
* Considera mayúsculas y minúsculas al definir los patrones.
